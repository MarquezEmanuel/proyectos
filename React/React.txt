
Comandos

npx --version
npx create-react-app nombreAplicacion
npm start

Estructura de archivos:

- node_modules: Nucleo de Node
- public: Mantiene los archivos estaticos
- index.html: Contiene el root y es donde se renderiza la aplicacion
- robots.txt: Es para los rastreadores del navegador
- src: Es la carpeta donde se trabaja
